# Sistema-de-Gesti-n-de-Libros-Electr-nicos
Este sistema permitirá administrar libros digitales de manera organizada, facilitando el registro, búsqueda, consulta y control de libros electrónicos dentro de una plataforma.
Objetivo del sistema
Desarrollar un sistema de gestión de libros electrónicos utilizando programación funcional, que permita administrar información de libros digitales mediante módulos organizados y eficientes.
Objetivos específicos
Registrar libros electrónicos.
Consultar libros almacenados.
Buscar libros por autor, categoría o título.
Editar información de libros.
Eliminar registros de libros.
Gestionar usuarios del sistema.
Mantener una estructura modular y funcional del software.
